# 🚀 Guía Básica de Backend
## 🧠 Introducción al Backend

Lo primero que se debe hacer es escoger un lenguaje de programación. Así como existen varios lenguajes para el **frontend**, también los hay para el **backend**.

### Lenguajes populares:
- Go
- PHP
- Ruby
- Python
- Node.js
- TypeScript

### Entornos empresariales:
- C#
- Java

> ⚠️ El entorno y las herramientas cambian según el lenguaje elegido.

---

## 📚 Fundamentos básicos

Antes de empezar:

- Variables  
- Funciones  
- Listas (arreglos)  
- Clases y objetos  

> ✅ Con estas bases ya puedes empezar a crear proyectos.

---

## 🌐 Conceptos de HTTP

Conceptos clave:

- **Methods:** `GET`, `POST`, `PUT`, `DELETE`  
- **Request**  
- **Response**  
- **Status Codes:** `200`, `404`, `500`  
- **Headers**  
- **Cookies**
